Что нужно настроить перед тестированием:

В файле проекта,папка парсер, в ней сам скрипт. 
В нем нужно раскоментировать последнюю строчку,
которая отвечает за отправку телеграм сообщений
 ,а потом саму функции отправки

Для запуска проекта перейти в дерикторию:

$ cd Map_Parser

После активировать виртуальное окружение:

$ source venv/bin/activate

Для запуска сервера:

$ python manage.py runserver

Если возникают ошибки о том,что нет какой то библиотеки(модуля) прописать эту команду:

$ pip install -r requirements.txt

Сервер ,на данный момент находится в дебаг режиме. При установке его на сервер(Nginx или Apache) ,либо хостинг, надо сделать:

1) В файле .env строку DEBUG = True заменить на DEBUG = False
2) В файле Map_Parser/settings.py в пустой список ALLOW_HOSTS добавить значения хоста порта и тд
(при дебаг моде устанавливается ['.localhost', '127.0.0.1', '[::1]']),можно заменить на "*".(при деплойте на хостинг может сам заменяться на нужный)
3) В конфигурациях сервера настроить статические файлы(tempalates - Map_Parser/templates | static files - Map_Parser/static | media files - Map_Parser/media)
!!!Настроить медиа обязательно,так как без этого не будет работать скачка файлов!!!
